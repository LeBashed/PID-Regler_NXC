# PID-Regler in NXC

Schulisches Robotikprojekt zur Regelung eines Roboters mithilfe
eines PID-Reglers. Ziel war es, Sensordaten auszuwerten und eine
stabile Regelung zu erreichen.

Technologien:
- NXC (Not eXactly C)

Schwerpunkte:
- P-, I- und D-Anteil
- Sensorauswertung
- Regelparameter-Anpassung
